# booking

## PROJECT DESCRIPTION

### Booking + Bloqueos + Reprogramación + Auditoría. Una clínica permite reservar citas por día (sin hora). Cada proveedor tiene:
● días habilitados
● capacidad diaria
● fechas bloqueadas (vacaciones / no disponibilidad)
● regla de reprogramación con ventana
### El sistema debe permitir:
● reservar
● reprogramar
● cancelar
● listar disponibilidad
● auditar cambios


### Links
📌 Live site URL: [here](https://github.com/Tonyva002/booking)

### Built with

- Mysql
- Node js
- Express
- React js
- vite
- Axios
- Tailwindcss
- Docker
- MVVM
- Clean Architecture
- vitest
- testing-library
  

### Necessary tool:

- Visual Studio Code (VSC)
- Docker desktop

### Steps to execute:

- Step 1. Download or Clone the repository
  
- Step 2. Unzip the file if you have downloaded it
  
- Step 3. Open in Visual Studio Code (VSC)
  
- Step 4. In the Visual Studio Code terminal, run:
  
```bash
copy booking-backend\.env.example booking-backend\.env
```
- Step 5. Open docker
  
- Step 6. In the Visual Studio Code terminal, run:

```bash
docker compose up --build
```
- Step 7. In the browser:

```bash
http://localhost:5173
```
